## NER数据增强方法
1. 替换实体（修改自[nlpcda-github](https://github.com/425776024/nlpcda)）
  * 修改点：
    1. 因场景需求，部分实体是同一大类下的不同细分（如日期下包含签署日期和起始日期），这种情况做增强的时候可以将一大类的实体互相替换，同时也输出替换后的小类。
      * NER实体标注样例（BIO标注法）：B/I-实体大类_实体小类（如：B-TIME_sign）
